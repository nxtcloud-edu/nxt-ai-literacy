# WO-021: 간격 시스템 도입 및 카드·탭·버튼 리듬 정비
상태: 대기
작성: Claude (Planner) / 실행: Hermes (Coder)
워크트리 게이트: `wo/021` 브랜치

## 목표
/design review에서 확인된 간격 문제 해소 (사용자 리포트 + 실측). 임의 간격을 4px 배수
스케일로 통일하고 카드·탭·네비의 수직 리듬을 정비한다.

## 설계 결정 (변경 금지)
1. **spacing scale** (theme.css SSOT): `--sp-1:4px; --sp-2:8px; --sp-3:12px; --sp-4:16px;
   --sp-6:24px; --sp-8:32px` — 이후 마진·패딩·gap은 이 변수만 사용.
2. **카드 수직 리듬** (index·cohort 공통): 패딩 `--sp-6`, 라벨↓`--sp-2`, 제목↓`--sp-1`,
   메타↓`--sp-3`, 칩·좋아요 줄↓`--sp-3`, 날짜 줄. 날짜와 ↗ 아이콘은 같은 줄에서
   `align-items:center` 정렬, 하단 패딩 침범 금지.
3. **한국어 줄바꿈**: 제목·메타·코호트명에 `word-break:keep-all` (기존 overflow-wrap은
   URL 등 비한글 오버플로 대비로 유지).
4. **탭 그룹 구분**: 정렬 그룹과 분류 그룹 사이 `--sp-3`, 그룹 내 gap `--sp-2`.
   정렬 그룹 앞에 mono 라벨(`정렬`·`분류`) 소형 표기는 선택 — 넣을 경우 muted 색.
5. **터치 타깃**: `.filter`·정렬 버튼 `min-height:44px` (패딩으로 확보, 모바일 포함).
6. **네비 우측 높이 통일**: 테마 토글과 업로드 버튼 `min-height:40px`로 동일.
7. **눌림 피드백**: 모든 버튼·카드 `:active{transform:translateY(1px)}` (reduced-motion 시 생략).
8. **레거시 이름 중복 제거**: 카드 메타 줄에서 `title === name`이면 이름 생략(코호트만 표시)
   — index·cohort 렌더 함수.
9. 다크 모드 회귀 없음. 서버 코드·infra 수정 금지.
10. **커밋 분리** (최소 2): ① 간격 스케일·카드·탭 리듬 ② 줄바꿈·터치 타깃·중복 제거 교정.

## 작업 단계
1. theme.css 스케일 변수 + 컴포넌트 간격 정비 (4페이지 인라인 스타일의 임의 간격도 스케일로)
2. DRY_RUN 브라우저: 라이트/다크 × index·cohort·upload·view — 카드 리듬·탭 그룹·줄바꿈 확인
3. npm test 그린(회귀), TURN_LOG 완료 헤더 + wo/021 커밋
